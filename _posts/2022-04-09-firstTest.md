# 첫 번째 포스트 테스트

### GitHub Blog 를 Local 에서 확인하려면..

1. Ruby 설치
2. 관련 package 설치(jekyll 등)

Command with Ruby 실행 후, Blog Root Folder로 이동 후, 
아래와 같은 명령어를 써주면 된다.

```sh
@LocalBlogFolder > bundle exec jekyll serve --trace
```

서버가 실행되었으면, http://localhost:4000 으로 블로그 접속 가능.
